AnnotationApiLink describes output file link

## FileInfo example ##

```html
{
  "Href": "string",
  "Rel": "string",
  "Type": "string",
  "Title": "string"
}
```

## FileInfo fields ###

|Name|Description
|---|---
|Href|File download url
|Rel|Reserved
|Type|Reserved
|Title|File name
